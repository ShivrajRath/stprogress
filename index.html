stProgress.js

Check on github

A light-weighted slim top progress bar to indicate service calls happening from your app. This is particularly useful for AJAX heavy applications. Just include the library and it takes care of rest.

Install

  • stprogress.js
  • stprogress.min.js
  • npm install stprogress
  • bower install stprogress

How to use

  • Include the script.
  • Add <div id="progress"></div> to your primary html.
  • Move stprogress.css content to your css. You can customize progress bar color.
  • That's all. Read blog post

Features

  • No Configuration needed
  • Automatically displays progressbar if AJAX calls are in background
  • Runs with any framework
  • AMD/CommonJS Support
  • 1kb minified
